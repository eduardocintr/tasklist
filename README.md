
# 📱 Lista de Tarefas

---

## 🚀 Como rodar o projeto

### 1. Clone o repositório
```bash
git clone https://github.com/eduardocintr/tasklist.git
cd tasklist
```

### 2. Instale as dependências
```bash
flutter pub get
```

### 3. Rode no navegador
```bash
flutter run -d chrome
```

> 💡 Se tiver problemas com o Chrome, use:
```bash
flutter run -d web-server
```
